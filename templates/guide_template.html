사용 API 목록

  • 유기동물 정보 조회 서비스

    1. 시도,시군구 조회 오퍼레이션
      • 유기동물보호소가 위치한 전국단위 시,도,군 정보를 json형태의 데이터로 반환
        • 요청메시지: 없음
        • 응답메시지: resultMsg:결과메시지, orgCd:시도코드, orgdownNm:시도명

      • JSON 데이터예시: 전국에 있는 보호시설,대행업체 정보

    2. 유기동물상세정보 조회 오퍼레이션
      • 요청조건에 맞는 유기동물집합의 상세정보를 json리스트형태의 데이터로 반환
        • 요청메시지: bende:검색시작일, endde:검색종료일, upkind(축종코드), uprcd:시도코드, orgcd:시군구코드, state:공고상태, neuter_yn:중성화여부

      • JSON 데이터예시: 경상북도 경주에 위치한 유기동물中

    3. 유기동물보호 및 등록대행업체 정보조회 오퍼레이션
      • 요청한 정보에 맞는 유기동물보호 및 등록대행업체집합을 json리스트형태의 데이터로 반환
        • 요청메시지: addr: 검색주소, orgNm:시설명, pageNo: 페이지 번호, numOfRows: 한 페이지 결과 수
        • 응답메시지: resultMsg:결과메시지, addr:주소, addrDtl:상세주소, htel:핸드폰번호, memberNm:대표자명, email:이메일

      • JSON 데이터예시: 천안시에 위치한 유기동물 보호 및 등록대행업체中

  • Google Geocoding API

    1. Geocoding 정보조회 오퍼레이션
      • 검색한 주소의 좌표,경도,위도를 포함한 여러 정보를 json리스트형태의 데이터로 반환
        • 요청메시지: address:주소정보
        • 응답메시지: address components: 결과주소정보, formatted_address: 전체 주소정보, geometry: 위도,경도,좌표정보, status: 응답여부

      • JSON 데이터예시: "한국기술교육대학교" 검색결과

  • Kakao Maps API

    1. Kakao Map 생성
      • 지도를 생성한 뒤, 마커로 표시한 지역으로 이동,가리킨다.
        • 요청메시지: address: 주소정보
        • 응답메시지: result: 지도정보

      • 지도이미지: "강남구" 검색결과
